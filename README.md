# Stellar Sandbox

An interactive 3D solar system simulator built with React Three Fiber.

## Features

- **Realistic Physics**: N-body gravitational simulation
- **Celestial Events**: Eclipses, alignments, and day/night cycles
- **Science Tools**: True scale, distance labels, light speed visualization
- **Educational Guide**: Interactive tour of the solar system

## Getting Started

1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`
4. Open http://localhost:8080

## Technologies

- React
- Three.js (@react-three/fiber)
- Tailwind CSS
- Zustand (State Management)
- Framer Motion
